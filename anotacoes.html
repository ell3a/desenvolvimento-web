Capítulo 1: Blocos de Montagem de Páginas na Web

O HTML recebe o conteúdo das páginas web e descreve seu significado.

Toda página web é formada por três elementos

  • conteúdo de texto;
  • referência a outros arquivos;
  • marcadores: são elementos em HTML que descrevem o conteúdo texto

O HTML inclui informações sobre o significado conteúdo de uma página, essa informação é chamada de marcação. O significado do conteúdo de uma página é chamado de semântica. O HTML não define como o conteúdo deve ser apresentado.

O elementos a de HTML é o mais poderoso, pois é o que vincula uma página a outra, ou a outra parte da página. Um exemplo do uso da âncora para outro site na internet: Página GitHub.
